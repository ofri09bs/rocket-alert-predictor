# 🛡️ Iron Dome Tactical C2

Welcome to the intelligent alert prediction dashboard. 
This site presents an AI-based system that analyzes the real-time security situation on the ground, providing threat assessments and estimated time windows for incoming alerts (sirens) across the country.

## 🔍 How Does It Work?
The system learns from a vast historical dataset and does not rely on guesswork. It analyzes the field using several key metrics:
* **Operational Sector Isolation:** The system categorizes each city into its relevant sector (North, Center, South, or Negev). It isolates events, understanding that an extreme load in one sector does not necessarily endanger another.
* **Trend Identification (Acceleration):** The AI algorithm calculates the rate of events in real-time, distinguishing between a tense routine and continuous barrages that require immediate combat readiness.
* **Live Data Synchronization:** The system is synchronized with a live data stream to "feel" the field and adjust its tactical predictions to the operational reality at any given moment.

## 🕹️ System Capabilities
* **Accurate Threat Assessment:** Select a city or settlement from the list to receive an immediate tactical report, including the probability of an event and an estimated time window.
* **War-Game Simulation Mode:** Toggle the simulation switch to manually input national and regional alert loads. This tool allows you to test how the AI model reacts to extreme scenarios and heavy combat conditions.
